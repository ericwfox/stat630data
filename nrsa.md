# nrsa

National Rivers and Streams Assessment (NRSA) data set.  

Variable | Description
----------------|---------------------------------------------------
`lon` | Longitude
`lat` | Latitude
`cond` | Stream condition (Good, Fair, Poor)
`cond2` | Stream condition coded as 1=Poor, 2=Fair, 3=Good
`mmi` | Macroinvertebrate multimetric index (a score from 0-100)
